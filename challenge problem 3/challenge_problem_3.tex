\section*{Challenge Problem 3}
Let $X_1,X_2,X_3,X_3,X_4,X_5$ be i.i.d. random variables having a continuous distribution function. Then
$\pr{X_1>X_2>X_3>X_4>X_5|X_1=max(X_1,X_2,X_3,$ $X_4,X_5)}$


\section*{Solution}
In the world where $X_1$ is the max value, we simply need to find:
\begin{align}
    \pr{X_2>X_3>X_4>X_5}
\end{align}
Since each of the random variables and identical and independent of each other and have continuous distribution function.
\begin{align}
    f_{X_2}(x_2) = f_{X_3}(x_3) =f_{X_4}(x_4) =f_{X_5}(x_5)=f_X(x)
\end{align}
If we choose the values for each random variable then they can be arranged in $4!$ ways but only 1 way is there to put them:
\begin{align}
    X_2>X_3>X_4>X_5
\end{align}
the probability then would be:
\begin{align}
    \pr{X_2>X_3>X_4>X_5} &= \frac{1}{4!}\\
    &=\frac{1}{24}
\end{align}
